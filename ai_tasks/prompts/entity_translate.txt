任务：把 entity 名称翻译成简洁中文（≤10字名词性短语）。仅输出 JSON。

【规则】
- bug 类型 → 抓痛点（"广告问题"/"App 闪退"/"通知缺失"）
- feature 类型 → 抓功能（"首发阵容"/"夜间模式"/"小组件"）
- competitor → 保留品牌名（FotMob、Sofascore、ESPN，不强行音译）
- player / league → 中文标准译名（Cristiano→C 罗；La Liga→西甲）
- 输入已是中文 → 原样返回
- 已是合理短语 → 精炼即可，不扩写

【输入】
entity_type: {entity_type}
primary_name: {primary_name}
canonical_id: {canonical_id}

【示例】
- bug "ads" + bug_inappropriate_ads → {{"chinese_name":"不当广告"}}
- bug "Não está abrindo o app" + bug_app_not_opening → {{"chinese_name":"App 打不开"}}
- bug "sudden shutdown" + bug_crash → {{"chinese_name":"突然闪退"}}
- bug "list of daily events on home widget doesnt scroll anymore" → {{"chinese_name":"小组件无法滚动"}}
- feature "fantasy app" + feature_fantasy → {{"chinese_name":"梦幻体育"}}
- competitor "flashscore" → {{"chinese_name":"FlashScore"}}
- player "Zverev" → {{"chinese_name":"兹维列夫"}}

仅输出 JSON（无其他文字）：
{{
  "chinese_name": "≤10字中文短语"
}}
